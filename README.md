# Development
1. Start the application ```npm run dev```
2. Open the application: http://localhost:5173/

# Build
Always run a build before pushing code.
```sh
npm run build
```

# E2E Testing
## Prerequisites
Start the application.
## Run tests
```sh
npm run test:e2e
```
## After tests
Terminate the application.
